Escape

An action stealth VR game made with the Unity Engine.

About Escape

Escape is a VR stealth game with a small puzzle element. You start locked inside a jail cell with clues on how to escape. After solving the puzzles, with only a silenced pistol, you must stealth your way though rooms of enemies each patrolling, looking for you.

Escape focuses on providing an atmospheric game enviroment using both volumetric lighting and ambient sounds.

Escape was made with the Unity Engine, Steam VR and developed on the HTC Vive
